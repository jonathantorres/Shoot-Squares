- Shoot Squares -
A little example of making a canvas game with Easeljs.

Move the triangle with arrow keys and shoot with the spacebar.
Shoot all the squares on top, but avoid them as they fall.

You can play the example here: http://jonathantorres.com/examples/shoot_squares/shoot_squares.html

Read the blog post here: http://www.jonathantorres.com/blog/2012/02/game-example-with-easeljs/
